# get-socks5

A simple module that gets a free public SOCKS5 list, checks the working socks and writes them on file

## Requirements
* NodeJS version (`8.x.x`) or higher

## Installation
```sh
npm i get-socks5
```

## Usage (Examples)
You can find a simple example on the `test` folder
